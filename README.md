## Descripción del proyecto

En esta actividad se implementó un proceso de análisis y procesamiento de datos utilizando Apache Spark sobre la plataforma Databricks Community Edition.

Se trabajó con el dataset **Teen Mental Health**, el cual contiene información relacionada con hábitos de uso de redes sociales, sueño, actividad física, rendimiento académico y variables asociadas a la salud mental de adolescentes.

El proyecto incluyó las siguientes etapas:

* Análisis y comprensión del conjunto de datos.
* Diseño del esquema de almacenamiento.
* Definición del esquema mediante StructType en PySpark.
* Carga y lectura del archivo CSV en Apache Spark.
* Validación de calidad de los datos.
* Generación de identificador único para los registros.
* Persistencia de la información en formato Delta Lake.
* Validación de metadatos mediante Spark SQL.
* Consultas analíticas utilizando SQL y PySpark.
* Elaboración de visualizaciones para el análisis de los datos.

## Resultados obtenidos

Durante el análisis se identificaron patrones relacionados con la salud mental de los adolescentes, especialmente en variables como:

* Riesgo de depresión.
* Niveles de estrés.
* Niveles de ansiedad.
* Horas de sueño.
* Uso de redes sociales.

Las visualizaciones permitieron evidenciar relaciones entre estas variables y comprender mejor el comportamiento del conjunto de datos.
